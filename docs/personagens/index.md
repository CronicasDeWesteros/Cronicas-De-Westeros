# Personagens de Westeros

Esta seção reúne os **personagens históricos e políticos relevantes** do cenário atual e recente de Westeros.

Ela inclui:
- Governantes passados
- Figuras centrais das Casas
- NPCs de grande influência
- Líderes das facções ativas

Jogadores **não criam personagens a partir daqui**.  
Estas páginas existem como **referência canônica**.

> Nem todo nome muda o mundo.  
> Os que estão aqui, mudaram.
